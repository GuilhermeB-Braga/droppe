Interface **moderna e performática** para compartilhamento de arquivos temporários, construída com **Next.js 16**. Este projeto foca em uma **experiência de usuário** fluida, utilizando uploads paralelos e integração direta com Cloud Storage.


## Visão Geral

- Multi-upload: Seleção e envio de múltiplos arquivos simultaneamente via Promise.all.

- Validação de Arquivos: Filtros por tamanho (máx 10MB) e tipo de arquivo antes do envio.

- Feedback Visual: Estados de carregamento (isPending), mensagens de sucesso e ícones de status de download.

- Downloads Únicos: Controle visual de arquivos já baixados na sessão.
